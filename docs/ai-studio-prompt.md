# Промпт для Google AI Studio: Платформа Direct Buy (Landing + Admin UI)

**Контекст проекта:**
Я создаю высокоэстетичный веб-проект и административную панель для P2P-платформы "Direct Buy". Платформа помогает собственникам недвижимости срочно продать квартиру за 24 часа пулу закрытых инвесторов. Основной процесс сделки идет в Telegram-боте, но мне нужна **современная, сверхпремиальная Landing Page** для конверсии собственников/инвесторов и **чистая Admin Panel** для управления сделками.

**Технический стек (строгие требования):**
- Фреймворк: Next.js 15+ (App Router).
- Язык: TypeScript (Strict mode).
- Стилизация: TailwindCSS + `shadcn/ui`.
- Иконки: `lucide-react`.
- Анимации: `framer-motion` (тонкие, плавные микроанимации).

**ЗАДАЧА 1: Premium Landing Page (Главная страница `app/page.tsx`)**
Создайте потрясающий лендинг. Дизайн должен кричать "Эксклюзивность, Скорость, Большие Деньги".
- **Стиль:** Темная тема (Dark mode by default) или очень чистый минимализм со стеклянным морфизмом (Glassmorphism). Глубокие, дорогие цвета. Избегайте дешевых стоковых градиентов.
- **Hero-блок:** Мощный заголовок ("Срочный выкуп недвижимости напрямую инвесторами"). Кнопки Call To Action: "Продать за 24 часа" и "Стать Инвестором" (обе ведут на ссылку Telegram-бота).
- **Блок "Как это работает":** 3-4 шага с красивыми иконками (Заявка -> Аукцион -> Аванс -> Сделка).
- **Блок "Преимущества":** Отсутствие комиссий для продавца, жесткие SLA (аванс за 3 часа), безопасность (NDA).
- **Подвал (Footer):** Ссылки на публичные оферты, NDA и политику конфиденциальности.

**ЗАДАЧА 2: Admin Panel Layout & Dashboard (`app/admin/layout.tsx` и `app/admin/page.tsx`)**
Создайте современную структуру дашборда менеджера, используя `shadcn/ui` (Sidebar, Navbar, Cards, Tables).
- **Sidebar (Боковое меню):** Ссылки на "Дашборд", "Заявки (Аукционы)", "Инвесторы (Модерация)", "Настройки".
- **Dashboard Overview:** 
  - Карточки статистики (KPIs): "Активных аукционов", "Сделок за неделю", "Ждут проверки (Инвесторы)".
  - Таблица "Требуют внимания (SLA Alerts)": красные бейджи для сделок, зависших более чем на 2 часа (клиент молчит, инвестор не шлет аванс).

**ЗАДАЧА 3: Таблица Заявок (Сделок) (`app/admin/lots/page.tsx`)**
Детальный view всех объектов, используя `shadcn/ui` Data Table.
- **Колонки:** ID Лота, Адрес, Желаемая сумма, Текущий статус (Dropdown со статусами: Аукцион, Ожидает решения, Ждем доки, Готово к сделке), Таймер SLA, Действия (кнопка "Подробнее").

**Инструкции для ИИ (Как писать код):**
1. Выдай мне ПОЛНЫЙ, готовый к запуску код компонентов.
2. Не используй заглушки (без `// ... here goes implementation`). Пиши реальный, отзывчивый Tailwind-код.
3. Используй `shadcn/ui` компоненты (Card, Button, Badge, Table, Input). В ответе предполагается, что компоненты уже установлены в `components/ui/`.
4. Код должен быть абсолютно идеальным визуально. Добавь эффекты наведения (`hover:`), переходы (`transition-all`) и правильные отступы.
5. Данные для дашборда замокай статичными типизированными массивами.
